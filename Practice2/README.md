# Практична робота №2: Робота із форматом даних json, робота із запитами - метод fetch(). React

## Мета заняття
Ознайомитися з принципами роботи формату даних JSON, вивчити механізми виконання мережевих запитів за допомогою методу fetch() та опанувати створення інтерактивних веб-додатків на React.

## Виконані завдання
### Рівень 1
**Завдання 5:**
* Створіть файл HTML з елементом <script>.
* Використовуйте fetch для завантаження JSON-файлу з URL зображення.
* Створіть елемент <img> з URL зображення.
* Додайте елемент <img> на веб-сторінку.

### Рівень 2
**Завдання 5:**
* Напишіть програму, яка перевіряє, чи даний текст є дійсним JSON.
* Використовуйте вбудовані функції мови програмування або бібліотеки для перевірки JSON.
* Виведіть результат перевірки (дійсний JSON / недійсний JSON).

### Рівень 3
**Завдання 5:** Розробіть веб-додаток інтерактивну дошку завдань, на якій користувачі можуть додавати завдання та присвоювати їм теги або мітки.

### Рівень 4
**Завдання 5:** Доповніть ваш додаток можливістю реєстрації та управління профілем користувача (зміна пароля, інформації).

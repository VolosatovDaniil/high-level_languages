# Лабораторна робота №4: Взаємодія з базами даних

## Мета заняття
Поглибити знання щодо проектування реляційних моделей даних, опанувати принципи роботи ORM для маніпуляції даними та вивчити методи оптимізації продуктивності запитів. Набути практичного досвіду в реалізації систем автентифікації та авторизації користувачів для захисту серверних ресурсів.

## Виконані завдання
### Рівень 1:
**Завдання 1:** Створіть базу даних для інтернет-магазину з п'ятьма таблицями: Користувачі, Товари, Замовлення, Категорії та Відгуки. Визначте зв'язки між таблицями, такі як "один-до-багатьох" та "багато-до-багатьох".

### Рівень 2:
**Завдання 1:** Використовуючи будь який відомий вам фреймворк, реалізуйте моделі для бази даних інтернет-магазину. Додайте методи для створення, оновлення та видалення об'єктів у кожній таблиці.

### Рівень 3:
**Завдання 1:** Оптимізуйте запити до бази даних інтернет-магазину за допомогою індексів та запитів з попереднім завантаженням.

### Рівень 4:
**Завдання 1:** Реалізуйте аутентифікацію та авторизацію користувачів у системі інтернет-магазину за допомогою JWT (JSON Web Tokens). Забезпечте доступ до ресурсів тільки для авторизованих користувачів.

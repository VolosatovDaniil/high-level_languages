# Практична робота №1: Основи Python та робота з даними

## Мета заняття
Ознайомитися з базовими можливостями мови Python: лінійні конструкції, розгалуження, цикли та функції.

## Виконані завдання
### Рівень 1:
**Завдання 1:** Створіть програму, яка виводить на екран від 1 до 10.

### Рівень 2:
**Завдання 1:** Напишіть програму, яка знаходить середнє значення з трьох чисел, введених користувачем.

### Рівень 3:

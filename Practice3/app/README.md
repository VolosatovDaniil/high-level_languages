# Практична робота №3: Мобільна розробка на Android з Kotlin

## Мета заняття
Ознайомитися з розробкою під Android і мовою програмування Kotlin.

## Виконані завдання
### Рівень 1
**Завдання 5:** Створіть форму з двома текстовими полями для вводу чисел. Реалізуйте кнопку, при натисканні якої добуток введених чисел виводиться на екран.

### Рівень 2
**Завдання 5:** Створіть гру «вгадай слово».

### Рівень 3
**Завдання 5:** Підрахуйте кількість окремих слів в заданому користувачем файлі.

### Рівень 4
**Завдання 5:** Знайдіть однакові файли в заданій папці якщо вони є. Видаліть дублікати.

# Лабораторна робота №1: Python та Django

## Мета заняття
Ознайомитися з архітектурою веб-фреймворку Django, навчитися створювати проекти, додатки.

## Виконані завдання
### Рівень 1:
**Завдання 5:** Написати сайт курсу валют, в якого буде можливість додавати нові курси співвідношення валюти до гривні. Для кожної валюти має бути два значення курсу купівлі та курсу продажу. В результаті отримуємо таблицю із валютами, яка буде мати 3 колонки: назву, курс купівлі та курс продажу.

### Рівень 2:
**Завдання 5:** Додайте можливість збереження результатів поточного дня в окремій вкладці.

### Рівень 3:
**Завдання 5:** Додайте можливість додавання нових валют за допомогою вбудованою адмін-панелі.
